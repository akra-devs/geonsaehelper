## 상태/빈 화면
- 네트워크 오류: 에러 상태 카드 + 재시도 버튼
- Empty: 저장된 결과 없음 → 첫 판정 유도

## 접근성/터치 규칙(요약)
- 터치 타깃 ≥ 44×44, 본문 대비 4.5:1 이상, 포커스 이동 순서 정의
